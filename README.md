# 🩺 MammoCAD — AI-Powered Mammogram Analysis System

A full-stack clinical decision support system for mammogram analysis using CNN + Streamlit.

---

## 📁 Project Structure

```
mammogram_cad/
├── app.py                  # Main Streamlit app (login, routing, dashboards)
├── train_ai.py             # CNN training on CBIS-DDSM dataset
├── database.py             # SQLite database management
├── predict.py              # Inference + feature extraction
├── report_generator.py     # PDF report generation
├── visualizations.py       # All chart/plot helpers
├── auth.py                 # Auth helpers
├── config.py               # Paths, constants
├── assets/
│   └── style.css           # Netflix-dark custom CSS
├── models/
│   └── mammogram_cnn.h5    # Trained model (generated by train_ai.py)
├── data/
│   └── (CBIS-DDSM images go here — see below)
├── requirements.txt
└── README.md
```

---

## 🚀 Setup Instructions

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Prepare your CBIS-DDSM dataset

Place your locally downloaded CBIS-DDSM images like this:
```
data/
├── train/
│   ├── benign/       ← benign mammogram images (.png or .jpg)
│   └── malignant/    ← malignant mammogram images
└── test/
    ├── benign/
    └── malignant/
```

> If your dataset is organized differently (e.g., with CSV metadata), run:
> `python prepare_dataset.py` — this script reads the CBIS-DDSM CSV files and organizes images.

### 3. Train the model
```bash
python train_ai.py
```
This saves `models/mammogram_cnn.h5` after training.

### 4. Initialize the database
```bash
python database.py
```

### 5. Run the app
```bash
streamlit run app.py
```
